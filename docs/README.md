# ninout Documentation

This folder contains the project documentation for `ninout`.

- `docs/architecture.md`: module organization and responsibilities.

- `docs/execution-and-logs.md`: how to run DAGs and interpret run artifacts.
- `docs/examples.md`: practical examples based on repository transformations.
- `docs/development.md`: local environment, tests, and contribution guidance.
